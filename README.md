## How to use it
- Clone the repo or download the file material-design-colors.itermcolors
- iTerm2 > Preferences > Profiles > Colors Tab
- Click Color Presets...
- Click Import...
- Select the awesome_maty.itermcolors file
- Select the awesome_maty from Load Presets...

## the below tab
